# ApkUrlGrep
Tool that allow extract endpoints from APK files


## Install

```
▶ go get -u github.com/ndelphit/apkurlgrep
```

## Usage


```
▶ apkurlgrep -a ~/path/to/file.apk
Result of URLs:

https://example.com
https://example.net
https://example.edu

Result of URLs Paths:

/example
/admin
/onboarding
```
